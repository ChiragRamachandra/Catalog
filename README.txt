To run this final project

1.Extract the .zip file and place it in the vagrant environmet

2.Navigate to the Catalog directory inside the vagrant environment

3. run database_setup.py to create the database (ensure database_setup.pyc is created)

4. If you want pre-populated values run lotsofmenus.py to populate the database (works otherwise also)

5. run project.py and navigate to localhost:5000 in your browser
